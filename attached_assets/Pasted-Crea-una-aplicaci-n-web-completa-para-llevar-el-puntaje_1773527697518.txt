Crea una aplicación web completa para llevar el puntaje de un juego de mesa entre varios jugadores. La aplicación debe sentirse como una app profesional de contador de puntos, optimizada para celulares y diseñada para usarse fácilmente durante una partida real.

Debe poder instalarse como aplicación en el teléfono, funcionar sin internet y poder compartirse mediante un enlace para que otros usuarios también puedan abrirla e instalarla.

Además debe tener un modo opcional para que varios jugadores puedan ver el mismo marcador en tiempo real desde sus propios celulares.

La experiencia debe sentirse como una app moderna de juego.

---

# Tecnologías

Usar:

HTML  
CSS  
JavaScript moderno

React + Vite

Usar también:

Tailwind CSS

Organizar el proyecto con una estructura clara de componentes.

# Progressive Web App (PWA)

La aplicación debe implementarse como Progressive Web App instalable.

Crear:

manifest.json  
service-worker.js

Características:

- funcionar completamente offline
- poder instalarse en Android y iPhone
- abrir en pantalla completa
- cachear archivos para uso offline
- iconos de app
- splash screen

Agregar botón visible:

"Instalar aplicación"

Implementar evento:

beforeinstallprompt
